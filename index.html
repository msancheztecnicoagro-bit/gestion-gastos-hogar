# 💰 Gestor de Gastos Personales ![React](https://img.shields.io/badge/react-%2320232a.svg?style=for-the-badge&logo=react&logoColor=%2361DAFB) ![TailwindCSS](https://img.shields.io/badge/tailwindcss-%2338B2AC.svg?style=for-the-badge&logo=tailwind-css&logoColor=white) ![Firebase/Supabase](https://img.shields.io/badge/BaaS-Firebase%20%7C%20Supabase-orange?style=for-the-badge) Una aplicación web de gestión financiera personal minimalista e intuitiva, inspirada en YNAB y Mint. Permite a los usuarios registrar transacciones, visualizar sus gastos por categoría y mantener el control de su presupuesto mensual. ## ✨ Características Principales - **Dashboard General:** Resumen en tiempo real del saldo actual, ingresos y gastos del mes. - **Gestión de Transacciones (CRUD):** Añade, visualiza, edita y elimina ingresos o gastos fácilmente. - **Categorización:** Etiqueta tus transacciones (Comida, Transporte, Ocio, etc.). - **Visualización de Datos:** Gráficos interactivos para entender tus hábitos de consumo. - **Diseño Responsive:** Interfaz "Mobile-first" optimizada para cualquier dispositivo. ## 🛠️ Tecnologías Utilizadas - **Frontend:** React.js (Vite) - **Estilos:** Tailwind CSS - **Gráficos:** Recharts / Chart.js - **Backend/Base de Datos:** Firebase / Supabase ## 🚀 Instalación y Configuración Local Sigue estos pasos para ejecutar el proyecto en tu entorno local: 1. **Clona el repositorio:** ```bash git clone [https://github.com/TU-USUARIO/gastos-app.git](https://github.com/TU-USUARIO/gastos-app.git) cd gastos-app
